# 职场权力人格测评 H5 项目

## 🚀 已完成功能

### ✅ 核心功能
1. **完整测评流程**：引导页 → 测评页（30题随机抽取）→ 结果页 → 分享页
2. **六种人格类型**：支配者/服从者/施压者/承压者/切换者/安全词持有者
3. **120题题库**：权力偏好/压力应对/关系模式三维度各40题，池化设计
4. **结果分析**：详细的人格描述、职场建议、发展路径

4. **PWA 支持**：manifest.json + Service Worker 离线缓存
5. **管理后台**：题库可视化增删改查、导入导出、数据面板

## 🔧 部署方式

### 立即部署（三选一）

**1. Vercel（推荐，最快）**
```bash
# 安装 Vercel CLI
npm i -g vercel

# 部署
cd h5
vercel --prod
```

**2. Netlify**
- 将项目拖到 netlify.com
- 设置发布目录为 `src`

**3. GitHub Pages**
- 推送到 GitHub 仓库
- 设置 Pages 源为 `src` 目录

### 访问地址
部署后可通过以下方式访问：
- **主页面**：`https://your-domain.com`
- **管理后台**：`https://your-domain.com/admin.html`
- **二维码**：自动生成，支持微信扫码
