# 职场权力人格测评 - H5 项目

## 项目文件清单

```
h5/
├── README.md                    # 项目说明
└── src/
    ├── index.html               # 主页面（单文件包含所有页面逻辑）
    └── assets/
        ├── config.json          # 人格配置与类型定义
        └── questions.json       # 完整题库（120题，3维度×40题）
```

## 设计理念

### 视觉风格
- **深色沉浸式**：暗色背景+金属质感，营造"权力场"氛围
- **金+暗红主色**：传递权威感与张力
- **毛玻璃卡片**：现代科技感，内容层次分明
- **发光按钮**：引导操作，增强互动感

### 文案策略
- 直击职场痛点：控制欲、压力、边界感
- 六种人格各有一句"金句"，强化记忆点
- 结果页包含优势+风险+建议，深度打动用户
- 分享文案自带社交货币

## 题库扩展指南

题库存储在 `assets/questions.json`，按以下结构扩展：

```json
{
  "questions": {
    "power": [
      {
        "id": 121,                           // 新题目ID（递增）
        "text": "你的题目文本",
        "options": [
          "选项1（对应支配者）",               // 选项顺序固定对应6种人格
          "选项2（对应服从者）",
          "选项3（对应施压者）",
          "选项4（对应承压者）",
          "选项5（对应切换者）",
          "选项6（对应安全词持有者）"
        ]
      }
    ]
  }
}
```

**扩展原则**：
1. 三个维度(`power`/`stress`/`relation`)各增加题目
2. 选项按固定顺序对应6种人格类型
3. 随机抽取逻辑自动适配——每维度抽取固定数量（当前10题），修改 `slice(0, 10)` 中的数字即可调整

## 人格配置扩展

`assets/config.json` 存储人格类型定义，修改后结果页自动更新：

```json
{
  "personalities": {
    "NewType": {
      "id": "NewType",
      "name": "新类型名",
      "tagline": "一句话定义",
      "icon": "🔥",
      "description": "金句描述...",
      "traits": ["特质1", "特质2", "特质3"],
      "risks": ["风险1", "风险2"],
      "advice": "发展建议...",
      "color": "#FF5733"
    }
  }
}
```

## 部署方式

直接托管 `src/` 目录到任意静态服务器即可：

- **Vercel**：`vercel --prod`（免费）
- **Netlify**：拖拽部署（免费）
- **GitHub Pages**：推送即可
- **本地预览**：直接浏览器打开 `src/index.html`
